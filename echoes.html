Echoes in the dark

  • View Code

A game jam creation inspired by the horror game stifled. It is a puzzle game where sound is a key element to solving the puzzles. I used unreal engine for the programming, as it was my first proper game made in unreal I used blueprints. The full team can be seen on the itch.io page.

For this project I created every mechanic and system for gameplay from the ground up. This included object interaction, the visual shader, level logic and the puzzle mechanics. I created primitive tools to make level design easier and a zoning system to handle what areas were being processed for performance.
