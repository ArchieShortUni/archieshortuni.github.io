2d Java Game Project

This was one of my first university projects, a game where you control a swarm of particles through mouse clicks to navigate through basically obstical courses.

For this project I got to play around with some basic physics and maths to simular portals, gravity wells and very basic flocking. It has a full level editor where you can save and load created levels with all elements except portal, a few base game levels and a save system where it saves all the transformations and data of the current level mid play.
